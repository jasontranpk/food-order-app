## 	:toolbox: Getting Started

<!-- Prerequisites -->

<!-- Run Locally -->
### :running: Run Locally

Clone the project

```bash
  git clone https://github.com/jasontranpk/food-order-app
```

Go to the project directory

```bash
  cd food-order-app
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
